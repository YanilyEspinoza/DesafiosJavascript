Desafío Entregable 3


Definición de un algoritmo en JS que emplee array para agrupar elementos similares


1-. Abrir (DEVTOOLS).

2-. Chequear el array "Productos", debe estar vacío

3-. Puede agregar productos de la base de datos con la función "dataBase()"

4-. Los elementos pueden ser visualizados en forma de tabla mediante el uso de "console.table(Productos)"

5-. Si desea Cargar productos, puede hacerlo con la función "agregarProducto()", deberá indicar los datos solicitados por los mensajes emergentes del prompt.
